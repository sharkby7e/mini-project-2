CSS Snippet CheatSheet

A tool to copy and paste CSS for cool stylings

All code snippets shown were used to write this page!

Color Variables

This allows developers to use variables to adjust color throughout stylesheet

  :root {
    --primary : indigo;
    --secondary: lightblue;
    --card-primary: darkslategrey;
    --card-secondary: dimgrey;
   }

Media Queries

This detects width of screen and adjusts width of the container for the cards

  @media screen and (min-width: 992px) {
    .mediumBox{
      flex: 0 0 33.333%;
      max-width: 33.333%;
    }
  }

Linear-Gradient

Set's the background to a gradient with two colors

 body {
   background-image: linear-gradient(to bottom right, var(--primary), var(--secondary));
 }

Overlay Card with Title

This allows developers to overlay a container with a header

.smallBox h2{
  position: relative;
  top: -20px;
}

FlexBox Row

This creates a flexible container that sets its contained elements in line

.largeBox {
  display: flex;
  flex-wrap: wrap;
  flex-direction: row;
}

FlexBox Column

This creates a flexible container that sets its contained elements block style

.smallBox {
  display: flex; 
  flex-direction: column;
}
